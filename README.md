HTML5 APIs (german)

Vortrag vom 14. Mai 2013 auf der SAE Wien über (guess what) HTML5 APIs.


Benutzung

Um alles einfach nur anzusehen, die `html5apis.zip` downloaden, entpacken und
die index.html aufmachen.
